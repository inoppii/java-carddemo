# 詳細設計書: COTRN00C (Transaction List)

## 1. 処理概要
トランザクション履歴を一覧表示する。アカウントに関連付けられた取引情報を日時の降順（または昇順）で表示。

## 2. 入出力定義
- **画面 (BMS):** `COTRN00` (Mapset)
- **ファイル (VSAM):** `TRANSACT` (トランザクション履歴、KSDS)

## 3. 処理フロー
1. 指定されたカード番号またはアカウントIDに基づき、トランザクションファイルを検索。
2. `STARTBR` でブラウズを開始し、画面にリストアップ。
3. 詳細ボタンが押された場合、`COTRN01C` へ遷移。

## 4. 移行への考慮事項
- 大量データ対応: トランザクション履歴は件数が多いため、インデックス設計と効率的なページングが不可欠。
