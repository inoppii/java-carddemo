# 詳細設計書: COUSR00C (User List)

## 1. 処理概要
セキュリティファイル (`USRSEC`) に登録されている全ユーザーを一覧表示する。1画面に10件ずつ表示し、ページング（PF7/PF8）および特定のユーザーに対するアクション（更新・削除）の起点となる。

## 2. 入出力定義
- **画面 (BMS):** `COUSR00` (Mapset), `COUSR0A` (Map)
- **ファイル (VSAM):** `USRSEC` (ユーザーマスター)

## 3. 処理フロー
1. **ブラウズ開始:** `STARTBR` を実行し、指定されたキー（または先頭）から読み込みを開始。
2. **一覧取得:** `READNEXT` を最大10回繰り返し、一時テーブルに格納。
3. **ページング処理:**
    - `PF8` (Next): 次のページを表示。
    - `PF7` (Prev): 前のページを表示（通信領域に保存した開始キーを使用）。
4. **アクション遷移:** 行の選択 ('U'=Update, 'D'=Delete) があった場合、対象のユーザーIDを通信領域にセットし、`COUSR02C` または `COUSR03C` へ `XCTL`。

## 4. データベース操作
- `EXEC CICS STARTBR`: ブラウズ開始。
- `EXEC CICS READNEXT`: 順読み。
- `EXEC CICS ENDBR`: ブラウズ終了。

## 5. 移行への考慮事項
- **ページングの実装:** SQL の `LIMIT` / `OFFSET` またはカーソルを用いたモダンなページング方式へ変換。
