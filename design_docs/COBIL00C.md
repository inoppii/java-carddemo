# 詳細設計書: COBIL00C (Bill Payment)

## 1. 処理概要
請求書の支払い処理を行う。支払い金額を入力し、アカウント残高を更新する。

## 2. 入出力定義
- **画面 (BMS):** `COBIL00` (Mapset)
- **ファイル (VSAM):** `ACCTDAT` (UPDATE)

## 3. 処理フロー
1. アカウントの現在残高を表示。
2. 支払い金額を入力。
3. `READ UPDATE` でアカウント情報を取得。
4. 残高から支払い額を減算し、`REWRITE`。
5. 更新結果を表示。

## 4. 移行への考慮事項
- **トランザクション整合性:** 支払処理は金銭に関わるため、ACID 特性を保証する。移行後は DB トランザクション (`COMMIT`/`ROLLBACK`) を適切に管理する。
