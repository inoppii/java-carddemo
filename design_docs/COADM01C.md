# 詳細設計書: COADM01C (Admin Menu)

## 1. 処理概要
管理者ユーザー向けのメインメニュー画面。ユーザー管理やマスタ管理など、特権ユーザーのみが実行可能な機能リストを表示する。

## 2. 入出力定義
- **画面 (BMS):** `COADM01` (Mapset), `COADM1A` (Map)
- **通信領域 (DFHCOMMAREA):** `CARDDEMO-COMMAREA` (COCOM01Y)
- **定数定義:** `COADM02Y` (管理メニューオプション設定)

## 3. 処理フロー
1. **初期処理:**
    - `EIBCALEN = 0` の場合はサインオン画面へ戻る。
2. **メニュー表示:**
    - 管理者用オプション (`COADM02Y`) を取得して表示。
3. **キー入力判定:**
    - `ENTER` キー: オプション番号のバリデーション。
    - `PF3` キー: サインオン画面へ戻る。
4. **遷移処理:**
    - 入力番号の妥当性確認。
    - プログラム名が 'DUMMY' でないことを確認。
    - 成功時、対応する管理者用プログラムへ `XCTL` で遷移。

## 4. データベース操作
- なし。

## 5. 例外処理
- PGMIDERR: プログラムが見つからない場合のエラーハンドリング (`PGMIDERR-ERR-PARA`)。

## 6. 移行への考慮事項
- **管理者権限の管理:** RBAC (Role Based Access Control) を導入し、移行先のバックエンド（Node.js/Java）で厳密な認可チェックを行う。
